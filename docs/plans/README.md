# Plans

Durable implementation plans for deferred or multi-phase work. A plan lives here
only while it is resumable — deferred, blocked, or intentionally postponed. When a
plan is complete or retired, move it to `docs/site_history/` and promote any
still-current rules into the relevant `docs/memory/<topic>.md` file.

`tools/check_memory_system.py` requires every `*.md` plan in this directory
(other than this README) to be indexed below.
